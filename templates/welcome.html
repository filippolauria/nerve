About

Welcome to NERVE, the Network Exploitation, Reconnaissance & Vulnerability Engine

NERVE is designed to identify both short-term and long-term security vulnerabilities through one-time or continuous scan modes on a given network or host.

You can achieve this either programmatically via the API or through the Graphical User Interface.

How does it work?

  1. Configure your network or domain list (e.g., 172.15.0.0/24).
  2. Define the types of attacks to run (e.g., Brute Force).
  3. Run the assessment.
  4. Wait for the results!
